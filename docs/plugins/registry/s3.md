# @streetjs/plugin-s3

**Official Street Framework plugin: AWS S3 object storage.**

- **Category:** [Storage](/StreetJS/plugins/category/storage/)
- **Tier:** Official (signed manifest, dependency-free)
- **Version:** `v1.0.2`
- **npm:** [@streetjs/plugin-s3](https://www.npmjs.com/package/@streetjs/plugin-s3)

## Install

```bash
npm install @streetjs/plugin-s3
```

## Usage

Register the plugin with the StreetJS plugin host. See the [Plugin System](/StreetJS/plugins/) guide for the full registration, capability, and signature-verification model, and the package README on npm for plugin-specific configuration.

Browse more in the [Plugin Marketplace](/StreetJS/plugins/marketplace/) or other [Storage](/StreetJS/plugins/category/storage/) plugins.
