# Frontend & Views Plugins

Render HTML from StreetJS with dependency-free view engines and frontend integrations — server-rendered, interactive apps with typed controllers that return markup, no SPA or client build step required.

1 official Frontend & Views plugin, all installable from npm. See the full [Plugin Marketplace](/StreetJS/plugins/marketplace/).

| Plugin | Description | Version | Links |
|--------|-------------|---------|-------|
| [@streetjs/plugin-htmx](/StreetJS/plugins/htmx/) | Official StreetJS plugin: HTMX support — a dependency-free view engine (layouts, partials), HX-request detection, response-header helpers and CSRF form fields for server-rendered apps. | `v1.0.0` | [npm](https://www.npmjs.com/package/@streetjs/plugin-htmx) |
