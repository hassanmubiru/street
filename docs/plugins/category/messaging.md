# Messaging Plugins

5 official Messaging plugins for StreetJS. All are signed, dependency-free, and installable from npm. See the full [Plugin Marketplace](/StreetJS/plugins/marketplace/).

| Plugin | Description | Version | Links |
|--------|-------------|---------|-------|
| [@streetjs/plugin-kafka](/StreetJS/plugins/kafka/) | Official StreetJS plugin: Apache Kafka streaming (wraps the dependency-free core Kafka client). | `v1.0.2` | [npm](https://www.npmjs.com/package/@streetjs/plugin-kafka) |
| [@streetjs/plugin-nats](/StreetJS/plugins/nats/) | Official StreetJS plugin: NATS publish/subscribe messaging (dependency-free protocol client). | `v1.0.2` | [npm](https://www.npmjs.com/package/@streetjs/plugin-nats) |
| [@streetjs/plugin-rabbitmq](/StreetJS/plugins/rabbitmq/) | Official StreetJS plugin: RabbitMQ messaging (wraps the dependency-free core AMQP 0-9-1 transport). | `v1.0.2` | [npm](https://www.npmjs.com/package/@streetjs/plugin-rabbitmq) |
| [@streetjs/plugin-sendgrid](/StreetJS/plugins/sendgrid/) | Official Street Framework plugin: SendGrid email. | `v1.0.2` | [npm](https://www.npmjs.com/package/@streetjs/plugin-sendgrid) |
| [@streetjs/plugin-twilio](/StreetJS/plugins/twilio/) | Official Street Framework plugin: Twilio SMS. | `v1.0.2` | [npm](https://www.npmjs.com/package/@streetjs/plugin-twilio) |
